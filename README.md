# 🏁 GridStart

GridStart is a Formula-inspired reaction time game built with React Native.

Test your reflexes against racing start lights, beat your personal best, and compete with players worldwide through a global leaderboard.

---

## ✨ Features

- ⚡ Fast reaction time gameplay
- 🌍 Global leaderboard
- 🔐 Google Authentication
- 📈 Firebase Analytics
- 💥 Firebase Crashlytics
- 🏆 Personal best tracking
- 🌙 Dark mode UI
- 📱 Android & iOS support

## 🚀 Getting Started

### Clone the project

```bash
git clone https://github.com/<your-username>/gridstart.git
```

### Install dependencies

```bash
npm install
```

### iOS

```bash
cd ios
pod install
cd ..
```

### Run Android

```bash
npm run android
```

### Run iOS

```bash
npm run ios
```

---

## 🔥 Firebase

Create a Firebase project and enable:

- Authentication (Google)
- Firestore
- Analytics
- Crashlytics

Then add:

- `google-services.json`
- `GoogleService-Info.plist`
